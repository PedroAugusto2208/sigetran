Como Renderizar os Diagramas PlantUML

Os arquivos .puml contem os diagramas em formato PlantUML. Para gerar as imagens (PNG/SVG), siga um dos metodos abaixo:

Metodo 1 - PlantUML Online (mais facil)

  1. Acesse plantuml.com/plantuml/uml
  2. Copie o conteudo do arquivo .puml e cole no editor online
  3. O diagrama sera gerado automaticamente
  4. Clique com botao direito na imagem para salvar como PNG

Metodo 2 - Extensao do VS Code

  1. Instale a extensao "PlantUML" no VS Code
  2. Abra o arquivo .puml
  3. Pressione Alt + D para visualizar o diagrama
  4. Use Ctrl + Shift + P → "PlantUML: Export" para salvar
